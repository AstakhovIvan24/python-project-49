### Installation


# Clone the repository


```bash
$ git clone https://github.com/AstakhovIvan24/python-project-49.git
```

Use make (Makefile):

make package-install

```bash
	python3 -m pip install --user --force-reinstall  dist/*.whl
```

make install

```bash
	poetry install
```
